Etiquetas HTML

Las principales etiquetas de HTML son:

Etiquetas Definición
1 "body" Indica para el cuerpo del contenido.
2 "head" Muestra información sobre el documento de html en metadatos.
3 "a" Esta etiqueta sirve para insertar enlaces en el documento.
4 "strong" Sirve para resaltar una palabra en el documento colocandolo el texto en negrita
5 "br" Ayuda a que el texto sea mas leible colocando un salto de línea
6 "H1…H6" Indica la importancia del titulo dentro del contenido donde h1 es el titulo y h2 el subtitulo
7 "img" Muestra imagenes dentro del documento o contenido.
8 "ol" Se coloca para listas ordenadas.
9 "ul" Se coloca para listas desordenadas.
10 "li" Crea listas de elementos en el documento.
11 "p" Sirve para colocar texto dentro un parágrafo.
12 "span" Personaliza el estilo de una parte del texto indicado.

Estilos CSS

Los diferentes estilos que pueden aplicarse en CSS son:

Etiquetas Definición
1 Colors: Cambia el color del texto del contenido.
2 Backgrounds: Permite establecer propiedades relacionadas con el color e imagen de cada parte de la pagina.
3 borders: Establece el mismo estilo, grosor o archura de los bordes de los componentes.
4 margins: Se utiliza para el margen de un elemento.
5 text: Determina el formato del texto.
6 fonts: Establece de una sola vez los valores para todoas las propiedades.
7 Height: Establece la altura de un elemento.
8 width: Establece la anchura de un elemento.
9 links: Representa un elemento que aun no se ha visistado.
10 tables: Se utiliza para decorar las tablas.
11 position: Especifica como un elemento es posicionado en el contenido.
12 Overflow: Controla la conducta del contenido que no caben en el documento.
13 float: Se utiliza para alinear un elemento.
